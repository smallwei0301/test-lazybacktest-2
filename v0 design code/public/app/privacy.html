隱私政策

LazyBacktest 尊重並保護所有使用者的個人資料。本政策說明我們如何蒐集、使用與保存資料，以及你可以運用的權利。

我們蒐集哪些資料？

  • 瀏覽器端設定：你在回測 App 中輸入的策略參數、風險設定與儲存的策略，僅儲存在你的瀏覽器 localStorage。
  • 雲端資料：執行批量優化、滾動測試、AI 評估與社群貼文時，我們會將必要的參數與結果透過 Netlify Blobs 暫存，以便提供跨裝置讀取。
  • 匿名使用統計：我們使用 Google Analytics 與 Netlify Analytics 追蹤匿名流量，用於產品改善，不會識別個人身份。

資料如何被使用？

  • 提供回測、優化、滾動測試與 AI 模型所需的運算與快取。
  • 支援 社群討論 的留言顯示與同步。
  • 分析整體使用行為，優化使用者體驗與系統效能。
我們不會販售、出租或交換你的個人資料，也不會用於廣告投放。

資料儲存與安全

  • 本地策略資料保存在瀏覽器，清除快取或更換裝置會造成資料遺失。
  • 雲端資料存放於 Netlify Blobs，採用權杖保護並限制 API 存取。
  • 所有 API 呼叫都經過 HTTPS 加密，避免中途竊聽。

你的權利

  • 可以隨時刪除瀏覽器中的策略紀錄或社群貼文。
  • 若需要刪除雲端資料，請透過 寄信給我 告知，我們會在 7 個工作天內處理。
  • 停用瀏覽器 Cookie 仍可使用主要功能，但我們可能無法判讀匿名流量來源。

政策更新

若未來有調整，我們會在此頁面註明更新日期並以頁面公告提醒。持續使用 LazyBacktest 即表示你同意最新版的隱私政策。
