LazyBacktest 隱私政策

我們依循台灣《個人資料保護法》與 Google Ads 政策，向使用者揭露資料蒐集、處理與保護方式，確保您能安心使用 LazyBacktest 股票回測服務。

一、資料蒐集範圍

當您使用 LazyBacktest 時，我們可能會取得下列資訊：

  • 必要的聯絡資訊：包含 Email、暱稱，用於回覆問題或寄送系統更新通知。
  • 操作紀錄：回測參數、執行時間與錯誤訊息，協助分析系統效能與除錯。
  • Cookie 與分析工具：使用 Google Analytics 4 與自建事件追蹤，以了解常用功能並優化使用者體驗。

二、資料使用目的

我們使用上述資料以達成下列目的：

  1. 提供回測服務、策略儲存與通知功能。
  2. 改進產品體驗、排除系統異常，並於必要時通知您維護排程。
  3. 於未來導入 Google Ads 時，依規範調整廣告呈現，包含投放頻率控制與轉換追蹤。

三、Cookie 與追蹤技術

  • Cookie 僅用於儲存登入狀態、偏好設定與回測草稿，不會記錄敏感個資。
  • Google Ads 及 Analytics 可能使用第三方 Cookie 了解廣告成效，您可透過 Google Ads 設定 或瀏覽器設定停用個人化廣告。
  • 我們會於頁面顯示 Cookie 通知，並提供繼續使用即表示同意的說明。

四、第三方服務與資料共享

LazyBacktest 不會出售或出租使用者資料。僅於以下情況分享必要資訊：

  • 與 Netlify、FinMind 等合作夥伴交換最小化的 API 呼叫資訊，用於提供行情或托管服務。
  • 依司法機關或主管機關合法要求配合提供相關資料。
  • 於廣告主要求提供匿名統計資料時，僅分享去識別化後的使用行為。

五、資料安全措施

  • 採用 HTTPS 與 Netlify Functions 保護資料傳輸，重要變更另有權限管控。
  • 定期檢測 OWASP Top 10 風險，並於 GitHub 設置自動化審查以降低錯誤部署風險。
