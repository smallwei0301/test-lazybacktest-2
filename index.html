      逐窗指標

      Walk-Forward 評分遵循 TradeStation® Walk-Forward Optimizer 與 QuantConnect Walkforward 分析的實務門檻，並加入 Pardo (2014) 定義的 Walk-Forward Efficiency（WFE）。公式流程如下：

      1. OOS 指標正規化：年化報酬、Sharpe、Sortino、勝率與最大回撤分別映射至 0～1，採 0.35／0.25／0.20／0.10／0.10 的權重求得 OOS 品質。
      2. 統計可信度：計算 Probabilistic Sharpe Ratio 與 Deflated Sharpe Ratio，平均後轉為 0.5～1 的 StatWeight，並估算最短樣本需求（MinTRL）。
      3. 單窗分數：WindowScore = OOS 品質 × StatWeight，跨窗取中位數，再以 WFE 中位數（取 0.8～1.2 區間）進行調整取得 Walk-Forward 總分。
      4. 評級門檻：總分 ≥ 0.70、WFE 中位 ≥ 0.8、PSR≥95% 視窗比 ≥ 50%、DSR 中位 ≥ 0.7 判定「專業合格」；總分 ≥ 0.50、WFE 中位 ≥ 0.6、PSR≥95% 視窗比 ≥ 30% 判定「可進一步觀察」；其餘為「未通過建議調整」。若整體 DSR ≤ 0 會再下修一級。
